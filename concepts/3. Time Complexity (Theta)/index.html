Time Complexity in JavaScript

Theta Notation

Theta notation is a mathematical notation used to describe the tight bound of an algorithm's time complexity. It provides a way to analyze and compare the efficiency of algorithms by expressing how their runtime grows as the input size increases.

Common Time Complexities

  • Θ(1) - Constant Time
  • Θ(log n) - Logarithmic Time
  • Θ(n) - Linear Time
  • Θ(n log n) - Linearithmic Time
  • Θ(n^2) - Quadratic Time
  • Θ(2^n) - Exponential Time
  • Θ(n!) - Factorial Time

Examples

- Θ(1): Accessing an element in an array by index.

- Θ(log n): Binary search in a sorted array.

- Θ(n): Iterating through an array to find a specific element.

- Θ(n log n): Sorting an array using merge sort or quicksort.

- Θ(n^2): Nested loops, such as bubble sort.

- Θ(2^n): Recursive algorithms that solve problems by dividing them into smaller subproblems, such as the Fibonacci sequence.

- Θ(n!): Algorithms that generate all permutations of a set, such as the traveling salesman problem.
